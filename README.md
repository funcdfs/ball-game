- 点击 sign up 注册用户，在表单中填写用户名和密码，再次点击 sign up 注册完成
- 再次填入刚才注册使用的用户名和密码，点击 sign in 登录到游戏中
- 点击单人模式体验玩法， 多人模式需要三人可以开始
- q 点击之后，使用鼠标左键控制发射普通技能的方向
- f 点击之后，可以朝着鼠标左键点击的指定方向移动一段距离 
- 多人模式中回车即可输入文字 
- 使用 esc 退出聊天框
